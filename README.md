📄 Document Management System (Java + JDBC + Console Based Appliction)

A simple Document Management System built using Core Java, JDBC, and MySQL. This project allows users to select a file from their local computer using a file chooser and 
securely store it in a MySQL database as a BLOB (Binary Large Object). It demonstrates file handling, database connectivity, and BLOB operations in Java.

🚀 Features
📁 Select files from the local system using JFileChooser
💾 Store files in MySQL using JDBC
🗂 Save the original file name
🔒 Store files as LONGBLOB
📚 Demonstrates Java File Handling
🔗 JDBC Database Connectivity
⚡ Simple and beginner-friendly implementation


🛠 Technologies Used
Java (Core Java)
JDBC
MySQL
MySQL Connector/J
IntelliJ IDEA / Eclipse
JFileChooser (Swing)
